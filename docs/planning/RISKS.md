# Risks & Assumptions

Live bets the project is making, tracked until resolved. Distinct from
DECISIONS.md, which only logs choices already settled — this is for
things still open or unproven.

## <risk/assumption title>
**Status:** open | mitigated | materialized
**Bet:** what we're assuming to be true
**If wrong:** impact if the assumption fails
**Mitigation:** what reduces the risk, if anything
